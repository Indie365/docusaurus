In this page, we will see some of the tools that will help you to build your documentation site with Docusaurus.

## Frontmatters

```md
---
id: slash
title: Slash, Our Mascot!
hide_title: false
hide_table_of_contents: false
sidebar_label: Say Hello to Slash!
description: We named our mascot Slash because most documentations start with //
keywords:
  - docs
  - docusaurus
  - mascot
---
```
